math.abs ​

絶対値を返す

lua
math.abs (x)

説明 ​

x の絶対値を返します。

サンプルコード ​

lua
local x = -5
print(math.abs(x))  -- 5

この例では、-5の絶対値を計算して5を返しています。

互換性 ​

  • Lua 5.1

関連項目 ​

  • math.min
  • math.max
