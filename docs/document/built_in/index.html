ビルトイン関数 ​

RaiaEngineはLuaJITをスクリプト言語として採用しています。LuaJITはLua5.1との互換性を保ちつつ、必要に応じてLua5.2、5.3から機能を追加しています。また、FFIやJIT関係の関数など、LuaJIT独自の機能を追加しています。

LuaJIT独自の拡張がある関数には、互換性の欄に(*)の表記を追加しています。
